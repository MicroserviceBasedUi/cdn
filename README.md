# CDN for Microservice Talk

# Prerequisites
```
npm install yarn -g
```

# Build
On Windows:
```
./build.ps1 -target build
```

On OS X:
```
./build.sh --target build
```

# Run
This command will run the CDN on port 9000.

On Windows:
```
./build.ps1 -target start
```

On OS X:
```
./build.sh --target start
```
